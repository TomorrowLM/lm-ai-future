

## 分页列表查询


**接口地址**:`/dsb/yqarw/api/yqa/partner/employee/pageList`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "blackListFlag": 0,
  "keyword": "",
  "pageNo": 0,
  "pageSize": 0,
  "qyzzId": 0
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|yqaPartnerEmployeeReq|YqaPartnerEmployeeReq|body|true|YqaPartnerEmployeeReq|YqaPartnerEmployeeReq|
|&emsp;&emsp;blackListFlag|黑名单标识:0=未拉黑,1=已拉黑||false|integer(int32)||
|&emsp;&emsp;keyword|关键字||false|string||
|&emsp;&emsp;pageNo|||false|integer(int32)||
|&emsp;&emsp;pageSize|||false|integer(int32)||
|&emsp;&emsp;qyzzId|企业组织ID||false|integer(int64)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|PageResult«YqaPartnerEmployee对象»|
|201|Created||
|401|Unauthorized||
|403|Forbidden||
|404|Not Found||
