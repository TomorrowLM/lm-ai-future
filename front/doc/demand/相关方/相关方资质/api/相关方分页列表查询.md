

## 分页列表查询


**接口地址**:`/dsb/yqarw/api/yqa/partner/qualificate/pageList`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "expireDate": "",
  "issueDate": "",
  "keyword": "",
  "pageNo": 0,
  "pageSize": 0,
  "qyzzId": 0,
  "type": 0
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|yqaPartnerQualificateReq|YqaPartnerQualificateReq|body|true|YqaPartnerQualificateReq|YqaPartnerQualificateReq|
|&emsp;&emsp;expireDate|资质证书到期日期||false|string(date-time)||
|&emsp;&emsp;issueDate|资质证书发证日期||false|string(date-time)||
|&emsp;&emsp;keyword|关键字||false|string||
|&emsp;&emsp;pageNo|||false|integer(int32)||
|&emsp;&emsp;pageSize|||false|integer(int32)||
|&emsp;&emsp;qyzzId|企业组织ID||false|integer(int64)||
|&emsp;&emsp;type|资质证书类型:10=企业资质,110=特种作业证,120=职业资格证,130=安全培训证,140=专业技能证||false|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- |

|201|Created||
|401|Unauthorized||
